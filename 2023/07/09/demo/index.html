md语法演示

1. 标题 [数个 “#” + 空格 前置]

标题会在目录与大纲分级显示, 可以跳转.
在 Typora 中建议开启 严格模式, 即不应为 #标题, 应为 # 标题.
应该要手动补上空格, 使得 Markdown 语法在其他文本编辑器上兼容.

2. 强调 [用 “**” 或 “__” 包围]

或者选中想要强调的文字按下 Ctrl + B.

E.G.

欢迎报考吉林大学!

3. 斜体 [用 “*” 或 “_” 包围]

或者选中想要强调的文字按下 Ctrl + I.

E.G.
欢迎大佬来浇浇我各种知识

4. 删除线 [用 “~~” 包围]

E.G.
我宣布个事儿, 我是Sabiyary!

5. *高亮 [用 “==” 包围]

(注意: 此为扩展语法)

E.G.
==我喜欢黄色, 也喜欢绿色==

6. 单行代码 [用 “`” 包围]

E.G.
sudo rm -rf /* (没事别乱敲这个! )

“请输入管理员密码: (闪烁的光标)”

7.代码块[按三个 “`” 并敲回车]

E.G.

8. 引用 [">" + 空格 前置]

引用是可以嵌套的!

E.G.

18岁, 是学生.

学生特有的无处不在(恼)

9. 无序列表 ["-" 或 “+” + 空格 前置]

三种前置符都可以, 敲回车会自动补全, 可在 Typora 设置中调整补全的符号, 敲回车后按下 Tab 会缩进一级.
E.G.

  • 一颗是枣树
  • 另一颗还是枣树

10. 有序列表 [数字 + “.” + 空格 前置]

敲回车会自动补全, 敲回车后按下 Tab 会缩进一级.

E.G.
我来这里就为了三件事:

  1. 公平
  2. 公平
  3. 还是tm的公平!

11. *上标 [用 “^” 包围]

(注意: 此为扩展语法)

E.G.

C语言中 int 的上限是 2^31^ - 1 = 2147483647

12. *下标 [用 “~” 包围]

(注意: 此为扩展语法)

E.G.

H~2~O 是剧毒的!

13. *注释 ["[^]" 后置]

(注意: 此为扩展语法)

需要在文末写上注释对应的内容

E.G.

今日我们相聚于此, 是为了学习 Markdown 的使用, 它的教程对于全体「观众」而言, 值得足足两个硬币的支持鼓励![^1]

[^1]: 沃兹·基·硕德 改编自「公鸡」普契涅拉.

14. 链接 [常用 “[ ]” + “( )” 分别包围文本与链接]

(注意: 文内跳转为扩展用法)

支持网页链接与文内跳转, 按住 Ctrl 并 单击鼠标左键 即可跳转.

E.G.
bilibili

15. 任务列表 ["- [ ]" + 空格 前置]

用 x 代替 [ ] 中的空格来勾选任务列表. 在 Typora 中可以直接用鼠标左键单击勾选框.

E.G.
TodoList:

  • [x] 刷B站
  • [ ] 写代码
  • [x] 起床

16. 表格 [用 “|” 绘制表格边框]

第一行为表头, 并由第二行分割线决定对齐方式与长度, 第三行及之后即表格数据
E.G.

学号 姓名 年龄
114514 田所 24
1919810 浩三 25

18. 分割线 [按三个 “*” 或 “-” 或 “_” 并敲回车]

由于 * 与 _ 均会自动补全, 所以我觉得 - 最为方便.
E.G.

19. Emoji表情 [":" 前置]

(注意: 英文输入为扩展语法)

这个功能唯一的要求就是英语水平要高, 或者大概记得各个 Emoji 的英文名.

E.G.

:sweat_smile:
:drooling_face:
:clown_face:

对于其余普通的 Markdown 文本编辑器, 可以直接将 Emoji 表情复制进来, 这是直接硬编码的 (刻进DNA里)

E.G.

😅🤤🤡

用好这个功能可以让你的文本非常的可爱! 太抽象了
这里分享一个可以复制全Emoji的网站

20. 内联 HTML 代码 [用 “<> </>” 包围]

只要你会写, 你完全可以把 Markdown 当作 HTML 来写.

同时, .md 文件可以直接导出成一个网页.

下划线可以选中想要下划的文字按下 Ctrl + U.

E.G.

我不会 HTML 呜呜呜... 浇浇我
简单的文字居中也可以这样

我差点忘了还有下划线这东西…

21. 内联 LaTeX\LaTeXLATE​X 公式 [用 “$” 包围]

(注意: 部分编译器会不识别部分符号)

用 $ 包围为单条公式, 按下两个 $ 并敲回车即生成公式块.

E.G.

LaTeX\LaTeXLATE​X 是最好用的论文排版语言! 不信你看!

an+bn=cna^n+b^n=c^nan+bn=cn

x+x2+y=ki3−xm\sqrt{x} + \sqrt{x^{2}+\sqrt{y}} = \sqrt[3]{k_{i}} - \frac{x}{m} x​+x2+y​​=3ki​​−mx​
